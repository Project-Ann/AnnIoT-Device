# AnnIoT-Devices
Connect devices to AnnIoT

## How to install ? 
  
  ```bash
    $ sudo npm i -g --usafe-perm anniot-device
  ```

### Getting Started
  
  **Login your account**
  
  ```bash
  $ anniot --login 
  
  ```
  
  **Start service**
  
  ```bash
    $ anniot --serviceStart    
  ```

Thats it. Module do everything automatically
  
  ----
  
  
  **Signout**
  
  ```bash
  
  $ anniot --signOut 
  
  ```
